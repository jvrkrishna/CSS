To control the spacing between the marker and the text, use **padding**, **margin**, or the `::marker` pseudo-element instead. ### Example (Old Syntax) ```css ul{ marker-offset:25px; } ``` ### Modern Alternative ```css ul{ padding-left:40px; } ``` --- # 5. list-style (Shorthand Property) The **list-style** property is a shorthand property used to specify multiple list properties in a single declaration. ### Syntax ```css list-style: list-style-type list-style-position list-style-image; ``` ### Example ```css ul{ list-style:square inside; } ``` Equivalent to ```css ul{ list-style-type:square; list-style-position:inside; } ``` --- # 6. resize The **resize** property specifies whether an element can be resized by the user. > **Note:** The `resize` property works only on elements whose `overflow` property is not `visible` (for example, `
