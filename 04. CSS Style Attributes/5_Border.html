# CSS Border The **border** properties are used to add and customize borders around HTML elements. You can control the border's **color, style, width, and shape**. --- # 1. border-color The **border-color** property specifies the color of the border. ### Syntax ```css border-color:red; border-color:blue; border-color:#00ff00; border-color:rgb(255,0,0); ``` ### Individual Border Colors ```css border-top-color:red; border-right-color:blue; border-bottom-color:green; border-left-color:orange; ``` ### Example ```html

``` --- # 2. border-style The **border-style** property specifies the style of the border. ### Values | Value | Description | |--------|-------------| | none | No border | | solid | Solid line | | dotted | Dotted line | | dashed | Dashed line | | double | Double line | | groove | 3D grooved border | | ridge | 3D ridged border | | inset | 3D inset border | | outset | 3D outset border | ### Individual Border Styles ```css border-top-style:solid; border-right-style:dashed; border-bottom-style:dotted; border-left-style:double; ``` ### Example ```html

``` --- # 3. border-width The **border-width** property specifies the thickness of the border. ### Syntax ```css border-width:2px; border-width:5px; border-width:10px; ``` ### Individual Border Widths ```css border-top-width:2px; border-right-width:4px; border-bottom-width:6px; border-left-width:8px; ``` ### Example ```html
Border Width Example
``` --- # 4. border (Shorthand Property) The **border** property is a shorthand property used to specify the border width, style, and color in one declaration. ### Syntax ```css border:width style color; ``` ### Example ```css border:4px dotted red; ``` **Equivalent to** ```css border-width:4px; border-style:dotted; border-color:red; ``` --- # 5. inset and outset Borders ### inset Creates a border that makes the element appear pressed inward. ```css border:8px inset gray; ``` ### outset Creates a border that makes the element appear raised outward. ```css border:8px outset gray; ``` ### Example ```html

Inset Border

Outset Border

``` --- # 6. border-radius The **border-radius** property rounds the corners of an element. ### Syntax ```css border-radius:10px; border-radius:20px; border-radius:50px; ``` ### Example ```css .box{ border:4px solid red; border-radius:20px; } ``` --- # 7. Individual Corner Radius You can round each corner separately. ```css border-top-left-radius:20px; border-top-right-radius:30px; border-bottom-left-radius:40px; border-bottom-right-radius:50px; ``` ### Example ```css .box{ border:4px solid blue; border-bottom-left-radius:100px; } ``` --- # 8. Different Radius for Each Corner You can specify all four corner radii. ```css border-radius:100px 0px 100px 0px; ``` **Order** ``` Top Left Top Right Bottom Right Bottom Left ``` ### Example ```css .box{ border:4px solid red; border-radius:100px 0px 100px 0px; } ``` --- # 9. Elliptical Border Radius Use a slash (`/`) to specify **horizontal** and **vertical** radii separately. ### Syntax ```css border-radius:50px/100px; ``` ### Example ```css .box{ border:4px solid green; border-radius:50px/100px; } ``` --- # 10.
